# 🏦 Bank Loan Analytics Dashboard (Power BI)

## 📌 Project Overview

This Power BI dashboard analyzes bank loan data to help understand loan performance, customer behavior, repayment trends, and key business metrics through interactive visualizations.

---

## 🎯 Objectives

- Monitor total loan applications
- Analyze funded amount and amount received
- Compare Good Loans vs Bad Loans
- Track monthly loan application trends
- Analyze loan applications by state
- Analyze loan purpose and home ownership
- Monitor average interest rate and DTI ratio

---

## 🛠️ Tools Used

- Power BI
- Power Query
- DAX
- Excel / CSV

---

## 📊 Dashboard Features

### Summary Page
- Total Loan Applications
- Total Funded Amount
- Total Amount Received
- Good Loan vs Bad Loan Analysis
- Funded Amount vs Repayment
- Average Interest Rate
- Average DTI Ratio

### Overview Page
- Monthly Loan Applications
- Loan Applications by State
- Loan Applications by Purpose
- Home Ownership Analysis
- Loan Term Analysis
- Interactive Filters
